#### COMO USAR? 
```
+-----------------------------------------------+
|                                               |
| --------- Para executar o servidor ---------- |
|                                               |
| Na pasta raiz da aplicação executar os        |
| seguintes comandos:                           |
| $ make                                        |
| $ ./server N_Porta_conexão e tamanho_buffer   |  
|                                               |
| ---------- Para rodar cada cliente ---------- |
|                                               |
| Acesse um navegador de sua preferência e      |
| digite: localhost: N_Porta_conexão            |
|                                               |
+-----------------------------------------------+
```
