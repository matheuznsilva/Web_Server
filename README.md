#### COMO USAR? 
```
+----------------------------------------------------+
|                                                    |
| ------------ Para executar o servidor ------------ |
|                                                    |
| Na pasta raiz da aplicação executar os seguintes   |
| comandos:                                          |
| $ gcc main.c -o server                             |
| $ ./server N_Porta_conexão                         |  
|                                                    |
| ------------ Para rodar cada cliente ------------- |
|                                                    |
| Acesse um navegador de sua preferência e digite:   |
| localhost: N_Porta_conexão                         |
|                                                    |
+----------------------------------------------------+
```
